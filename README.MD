# prisma-nest

Generate nestJS components using prisma types.


## Install

```bash
npm i -D prisma-nest
```

## Usage

```bash
npx prisma-nest <Enter the class / object name>
```

OR

```bash
npx gen <Enter the class / object name>
```
## Example

```bash
npx prisma-nest gen User
```

OR

```bash
npx gen User
```
