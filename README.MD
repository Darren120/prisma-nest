# prisma-nest

Generate nestJS components using prisma types.


## Install

```bash
npm i -D prisma-nest
```

## Usage

```bash
npx prisma-nest 
```

OR

```bash
npx gen 
```

This will give you a choice to generate either Controller, Module, Service, or all three nestJS componenets.
